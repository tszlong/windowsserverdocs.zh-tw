# <a name="add-host-information-to-the-hgs-configuration"></a>將主機資訊新增至 HGS 設定

>適用于： Windows Server 2019、Windows Server (半年通道) 、Windows Server 2016

從網狀架構系統管理員取得主機的相關資訊，並將其新增至 HGS。 此步驟會根據證明的模式而有所不同：

- [新增系統管理員信任證明的主機資訊](guarded-fabric-add-host-information-for-admin-trusted-attestation.md)
- [新增適用于 TPM 的主機資訊-受信任的證明](guarded-fabric-add-host-information-for-tpm-trusted-attestation.md)
