# <a name="vol"></a>vol

顯示磁片區標籤和序號（如果有的話）。  如果使用時不含 **參數，則** 會顯示目前磁片磁碟機的資訊。

## <a name="syntax"></a>語法

```
vol [<drive>:]
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
|--|--|
| `<drive>:` | 指定包含您要顯示磁片區標籤和序號之磁片的磁片磁碟機。 |
| /? | 在命令提示字元顯示說明。 |
