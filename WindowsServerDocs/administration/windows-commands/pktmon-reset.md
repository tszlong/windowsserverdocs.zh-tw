# <a name="pktmon-reset"></a>pktmon 重設

> 適用于： Windows Server (半年通道) 、Windows Server 2019、Windows 10、Azure Stack HCI、Azure Stack Hub、Azure

將計數器重設為零。

## <a name="syntax"></a>Syntax

```
pktmon reset [-counters]
```
