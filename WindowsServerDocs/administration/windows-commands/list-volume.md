# <a name="list-volume"></a>list volume

顯示所有磁碟上的基本和動態磁碟區清單。

## <a name="syntax"></a>語法

```
list volume
```

### <a name="examples"></a>範例

若要列出偵測到的磁片區，請輸入：

```
list volume
```

若要刪除 *磁片區 2*，請輸入：

```
delete volume=2
```
