# <a name="verify"></a>驗證

告訴命令提示字元工具 ( # A0) 是否要確認是否已正確地將檔案寫入磁片。

## <a name="syntax"></a>語法

```
verify [on | off]
```

### <a name="parameters"></a>參數

| 參數 | 描述 |
|--|--|
| `[on | off]` | 將 [ **驗證** ] 設定切換為開啟或關閉。 |
| /? | 在命令提示字元顯示說明。 |

## <a name="examples"></a>範例

若要顯示目前的 **驗證** 設定，請輸入：

```
verify
```

若要開啟 [ **驗證** ] 設定，請輸入：

```
verify on
```
